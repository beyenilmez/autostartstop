# Ping Management Actions

Actions for managing ping requests and customizing server list responses. These actions are for use with the [ping](/triggers/ping.md) trigger.

## Available actions
<!-- include-start -->
- **[deny_ping](deny-ping.md)**: Deny a ping request
- **[allow_ping](allow-ping.md)**: Allow a ping request to proceed normally
- **[respond_ping](respond-ping.md)**: Respond to a ping request with custom ping data
<!-- include-end -->
