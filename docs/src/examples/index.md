# Examples

This section contains real-world configuration examples that demonstrate how to use AutoStartStop in various scenarios.

## Available Examples

- **[Auto Start/Stop with Custom Ping Response](auto-start-stop-with-custom-ping-response.md)**: Demonstrates automatic server starting, stopping, and custom ping/MOTD responses based on server status.
- **[Scheduled Restart](scheduled-restart-with-proxy-start.md)**: Demonstrates automatic server starting on proxy startup and scheduled restarts using cron expressions.
- **[Hourly Random Reward](hourly-random-reward.md)**: Demonstrates scheduled command execution to reward random players with items using cron triggers.
